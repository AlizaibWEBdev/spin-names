Getting Started

How do I use the Wheel of Names for the first time?

Getting started with the Wheel of Names is simple and requires no registration:

  1. Enter your items: In the text box, add each name, prize, or option on a separate line
  2. Customize your wheel (optional): Change colors, spin duration, and text appearance
  3. Spin the wheel: Click the "SPIN THE WHEEL" button to start the random selection
  4. View results: The selected item will be displayed when the wheel stops

The tool works immediately in your web browser with no downloads required. You can use it on any device with an internet connection.

Is there a limit to how many items I can add to the wheel?

Yes, there are some practical limits to ensure optimal performance:

  • Recommended maximum: 100 items for best visibility and performance
  • Absolute maximum: 200 items (text may become hard to read)
  • Minimum: At least 2 items for the wheel to function

If you need to work with more than 200 items, we recommend:

  1. Dividing your list into multiple wheels
  2. Using our "Save Wheel" feature to create specialized wheels
  3. Running multiple rounds of selection if needed
Do I need to create an account to use the Wheel of Names?

No, the Wheel of Names is completely free to use without any account creation or registration. We designed it this way to:

  • Provide instant access to all users
  • Respect your privacy (we don't collect personal data)
  • Keep the tool simple and easy to use

All customization and spinning happens directly in your browser. If you want to save a particular wheel configuration, you can:

  1. Use the "Save Wheel" button to generate a shareable link
  2. Bookmark the page in your browser
  3. Copy the list of names/items to a text file for later use

Features & Usage

What are some creative ways to use the Wheel of Names?

The Wheel of Names is versatile for both personal and professional use. Here are some creative applications:

In Education:

  • Cold calling: Randomly select students to answer questions
  • Group assignments: Assign students to project teams fairly
  • Reading order: Determine who reads next in class
  • Reward system: Spin for classroom privileges or small prizes

For Businesses:

  • Contest winners: Select raffle or giveaway winners transparently
  • Meeting participation: Choose who speaks next in brainstorming
  • Task delegation: Assign weekly office chores fairly
  • Training exercises: Randomly select scenarios for role-playing

Personal Use:

  • Family decisions: Choose movie night picks or dinner options
  • Game night: Determine turns or select challenges
  • Fitness motivation: Randomly select workout exercises
  • Travel planning: Decide on vacation activities or destinations
How does the Wheel of Names ensure fair random selection?

Our Wheel of Names uses advanced algorithms to guarantee truly random and fair selections:

Technical Implementation:

  • Browser-based randomization: Uses JavaScript's cryptographically secure random number generator
  • Weighted distribution: Each segment has equal probability regardless of wheel size
  • Deceleration algorithm: Natural stopping pattern that doesn't favor any position

Fairness Features:

  • No memory: Each spin is completely independent of previous spins
  • Visual verification: Users can watch the physical spin and stopping point
  • Transparent process: No hidden weights or preferences in the code

For situations requiring absolute fairness (like official contests), we recommend:

  1. Recording the spin session for verification
  2. Having multiple witnesses observe the selection
  3. Using the "Share" feature to send results to all participants

Customization Options

Troubleshooting

Why isn't my wheel spinning when I click the button?

If your wheel isn't spinning, try these troubleshooting steps:

Common Issues and Solutions:

  • No items entered: Ensure you've added at least 2 names/items to the wheel
  • Browser issues: Try refreshing the page or using a different browser
  • Ad blockers: Disable any ad blockers or script blockers that might interfere
  • Slow internet: Wait a moment for all assets to load completely

Advanced Troubleshooting:

  1. Clear your browser cache and cookies for our site
  2. Check if JavaScript is enabled in your browser settings
  3. Try using the Chrome or Firefox browser for best compatibility
  4. Contact us with details about your device and browser version

If the problem persists, you can try our basic version of the wheel which has fewer features but more reliable performance on older devices.

How do I recover a saved wheel configuration?

If you've saved a wheel configuration but can't access it, follow these steps:

Recovery Methods:

  • Bookmark: Check your browser bookmarks if you saved the link
  • Browser history: Search your history for "spin-names.vercel.app"
  • Email: Look for messages where you might have shared the link
  • Local storage: Some configurations may be saved in your browser's local storage

Prevention for Next Time:

  1. When saving, email the link to yourself as backup
  2. Bookmark the page in multiple browsers
  3. Take a screenshot of your wheel settings
  4. Copy the list of names to a separate document

Note: For privacy reasons, we cannot recover lost wheel configurations from our servers as all data is stored locally in your browser.
